**Ultimate Computer Vision System**



**A simple computer vision app that detects faces, counts blinks, tracks hands, and recognizes emotions in real-time.**



**Quick Start (Windows)**



**1. Download all files to a folder**

**2. Run setup.bat (double-click it)**

**3. Run run.bat (double-click it)**



**That's it! The app will start with your webcam.**



**Controls**



**Press these keys while the app is running:**



**- Q or ESC - Quit**

**- R - Reset blink counter**

**- F - Toggle fullscreen**

**- I - Show/hide info**

**- S - Save screenshot**



**What It Does**



**- Face Detection - Finds faces and draws boxes**

**- Blink Counting - Counts every eye blink**

**- Hand Tracking - Counts fingers (1-5)**

**- Emotion Recognition - Detects Happy, Sad, Angry, Surprised, Neutral**

**- Performance - Shows live FPS and stats**

**Need Help?**



**Camera not working?**

**- Close Zoom/Teams/Skype**

**- Check Windows Privacy Settings > Camera**

**- Restart computer**



**App crashes?**

**- Run fix\_numpy.bat**

**- Make sure you're connected to internet for first run**



**Low FPS?**

**- Edit config.json, change resolution to \[640, 480]**

**- Close other programs**

**- Use better lighting**
